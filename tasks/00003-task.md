# Обновление правил принятия задач через [accepted]

Важно: выполнение плана из этого файла запрещено, пока в начале файла не будет добавлена строка `[accepted]`.

## Контекст
- Требуется жёсткое правило: выполнять задачу можно только если в начале файла есть `[accepted]`.
- Перед выполнением задачи нужно перечитывать файл, так как он мог быть изменён.
